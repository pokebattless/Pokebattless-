# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2026-07-02

### Added

- Initial release of PokeBattles — Cyber Arena
- Home screen with animated logo and online counter
- Matchmaking / lobby flow (ranked & casual)
- Turn-based battle system with HP bars, moves, and type effectiveness
- Hit particle effects and floating damage numbers
- AI-generated battle narration via the Claude API
- Match history and results overlay
