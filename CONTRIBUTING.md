# Contributing to PokeBattles

Thanks for your interest in contributing! 🎉

## How to Contribute

1. **Fork** this repository
2. **Create** your branch: `git checkout -b feat/your-feature`
3. **Make your changes** to `index.html`
4. **Commit**: `git commit -m "feat: add your feature"`
5. **Push**: `git push origin feat/your-feature`
6. **Open a Pull Request**

## Ideas for Contribution

- 🐛 Bug fixes in battle logic or UI
- 🎨 New themes / visual polish
- ⚔️ New moves, Pokémon, or type interactions
- 📱 Mobile responsiveness improvements
- ♿ Accessibility improvements

## Reporting Issues

Found a bug or have a feature idea? [Open an issue](../../issues/new) and describe:

- What happened / what you expected
- Steps to reproduce (if a bug)
- Screenshots if relevant

## Code Style

This is a single-file vanilla HTML/CSS/JS project — please keep it dependency-free and try to match the existing formatting style.
